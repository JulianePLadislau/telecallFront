Two-Factor Authentication

2FA Two-Factor Authentication

2FA is a security procedure that ensures that 2 unique factors are required for the authorization of an action. The first factor is the user's password, and the second factor can be authenticated via a token, fingerprint detection, facial recognition, code sent via SMS, among others.


2FA allows you to:

  • Send a password via SMS, voice, or email for user authentication.
  • Add an extra layer of security beyond the personal password.
  • Provide greater security for users.

Strengthen the security strategy of your business.

Adding a recovery phone number to an individual account can block up to:




100% of automated bots,


99% of mass phishing attacks,


and 66% of targeted attacks.

How it works?


  • The user accesses your website or application and enters the registered password to log into their profile or complete a transaction.

  • Telecall receives the login attempt and requests the user to enter their phone number to authorize access.

  • After entering their number, Telecall sends the user a one-time-use PIN code via SMS, call, or email.

  • The user enters the code on the website or application to complete the verification process.

Benefits

  • Provide enhanced security for your customers.
  • Reduce cases of fraud and breaches and prevent unauthorized access to data.
  • Send OTP through multiple channels, including SMS, voice, or email.
  • Channel flexibility ensures that users can complete desired tasks even if they have an issue with one channel. For example, sending OTP via SMS, and in case of failure, sending OTP via voice call, and in case of another failure, sending it via email.
  • Simple and quick implementation API.
  • Intuitive platform that allows you to view usage reports by day, month, or year and search using various filter criteria.
